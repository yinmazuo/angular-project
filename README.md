# angular-project

###导语：
这是用angular写的简单调查问卷的管理系统，

技术栈： webpack + angular + es6 + scss。

使用了一个ng的日历插件，基于bootstrap，不过本项目并没有使用bootstrap。

###实践步骤：
* `npm install`
* `npm run build`
* `npm run start`
